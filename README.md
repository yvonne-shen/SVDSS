🌿 SVDSS — 永續視覺化決策輔助系統
Sustainable Visualization Decision Support System
> **Note:** This SVDSS refers to the *Sustainable Visualization Decision Support System* for interior design sustainability — distinct from the bioinformatics SVDSS (Structural Variant Discovery from Sample-specific Strings).
---
簡介 Overview
SVDSS 是一套專為台灣室內設計產業開發的永續決策輔助系統，協助設計師在設計階段實現綠建材合規評估與 ESG 碳排報告自動化。
SVDSS is an AI-powered sustainability decision support system for Taiwan's interior design industry, enabling automated green building material compliance assessment and ESG carbon reporting.
---
核心功能 Features
🏗️ Rgi 綠建材使用率評估（法規門檻 60%）
📦 綠建材認證資料庫（內政部建築研究所核定清冊）
🔌 SketchUp Plugin 整合
📊 碳排係數分析
📄 ESG 合規報告自動輸出
🚦 紅綠燈合規機制（即時判斷是否達標）
